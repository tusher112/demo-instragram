what is the difference between Local storage and session storage?

Local storage and session storage both are used for keep store data. when we use website the server dont know who we are. The main difference between session storage and local storage is local storage stored data for permanent but session storage is temporary and local storage can store approximately 5MB data and session storage also can store 5MB data.Session storage will be destroyed after closed the browser.Local storage never destroy save data. we can manually edit or delete local storage data.Local storage is like cookie and session storage is like session.
